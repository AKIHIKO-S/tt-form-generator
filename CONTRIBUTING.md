# 🤝 Contributing

このプロジェクトへの貢献を歓迎します！

## 🐛 バグ報告

[Issues](https://github.com/YOUR_USERNAME/tt-form-generator/issues/new) で以下の情報をお知らせください：

- ブラウザとバージョン
- 再現手順
- 期待した動作と実際の動作
- スクリーンショット（あれば）

## 💡 機能要望

`enhancement` ラベルを付けて Issue を立ててください。

## 🔧 プルリクエスト

1. Fork
2. ブランチを切る (`git checkout -b feature/amazing-feature`)
3. 変更をコミット (`git commit -m 'Add amazing feature'`)
4. プッシュ (`git push origin feature/amazing-feature`)
5. PRを作成

### 開発環境

```bash
npm install
npm run dev
```

### コーディング規約

- インデント: 2スペース
- セミコロン: 不要（Vite/React標準に従う）
- 変数命名: camelCase
- 日本語コメント可

### テンプレートを追加するには

`src/App.jsx` の `TEMPLATES` オブジェクトにエントリを追加し、PRしてください。
他の卓球大会だけでなく、テニス・バドミントンなどのラケット競技、また文化系の大会テンプレートも歓迎します。
