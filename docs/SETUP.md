# 📖 詳細セットアップガイド

このドキュメントでは、生成したフォームを実際に運用開始するまでの手順を詳しく説明します。

## 📋 全体の流れ

```
[1] 生成ツールでHTMLとGASコードを生成
       ↓
[2] Googleスプレッドシートを準備
       ↓
[3] GASをスプレッドシートに設置
       ↓
[4] ウェブアプリとしてデプロイ
       ↓
[5] フォームHTMLにURLを設定
       ↓
[6] Webサイトに埋め込み
       ↓
[7] テスト送信
```

所要時間：約15分

---

## Step 1: 生成ツールで2ファイル生成

1. ツールを開く: https://YOUR_USERNAME.github.io/tt-form-generator/
2. テンプレートを選ぶ → 基本情報 → カテゴリ → デザインを設定
3. 出力タブを開いて以下2つをダウンロード：
   - `form.html` （Webサイトに置く）
   - `Code.gs` （Googleスプレッドシートに置く）

---

## Step 2: Googleスプレッドシートを準備

1. https://sheets.google.com を開く
2. **空白のスプレッドシート** を作成
3. ファイル名を任意に設定（例：「会長杯2026 申込管理」）

---

## Step 3: GASをスプレッドシートに設置

1. メニューから **拡張機能** → **Apps Script** を選択
2. デフォルトの `function myFunction() {}` を全て削除
3. ダウンロードした `Code.gs` の中身を全てコピーして貼付け
4. 保存（`Ctrl + S` または `Cmd + S`）
5. プロジェクト名を「卓球大会申込システム」などに変更

### 初期化を実行

1. エディタ上部の関数セレクタで `initializeSystem` を選択
2. **▶ 実行** ボタンをクリック
3. 「権限を確認」 → 自分のGoogleアカウントを選択
4. 「このアプリは Google で確認されていません」が出たら：
   - **詳細** → **（プロジェクト名）に移動（安全ではないページ）** をクリック
   - **許可** をクリック
5. 「✅ 初期設定が完了しました」が表示されればOK

⚠️ **注意**：「Google で確認されていません」という警告は、自分で作ったスクリプトでは正常な表示です。第三者ではないので「許可」してください。

---

## Step 4: ウェブアプリとしてデプロイ

1. エディタ右上の **デプロイ** → **新しいデプロイ**
2. 種類を選択（歯車アイコン） → **ウェブアプリ**
3. 設定：

| 項目 | 設定値 |
|------|--------|
| 説明 | 任意（例: v1.0） |
| 次のユーザーとして実行 | **自分** |
| アクセスできるユーザー | **全員** |

4. **デプロイ** をクリック
5. 表示された **ウェブアプリのURL** を必ずコピーして保存
   - 例: `https://script.google.com/macros/s/AKfyc.../exec`

⚠️ このURLが「アクセスできるユーザー：全員」になっていないとフォームから送信できません。

---

## Step 5: フォームHTMLにURLを設定

1. ダウンロードした `form.html` をテキストエディタで開く
2. `Ctrl + F` で `YOUR_GAS_WEB_APP_URL_HERE` を検索
3. Step 4 でコピーしたURLに置換

```javascript
// 変更前
var GAS_URL = 'YOUR_GAS_WEB_APP_URL_HERE';

// 変更後
var GAS_URL = 'https://script.google.com/macros/s/AKfyc.../exec';
```

4. 保存

---

## Step 6: Webサイトに埋め込み

### Jimdoの場合

1. Jimdoの編集モードを開く
2. **コンテンツを追加** → **ウィジェット/HTML** を選択
3. `form.html` の中身を **全選択してコピー**
4. ウィジェットに **貼付け**
5. **保存して公開**

### WordPressの場合

1. ブロックエディタで **カスタムHTML** ブロックを追加
2. `form.html` の中身を貼付け

### スタンドアロンHTMLとして公開する場合

`form.html` をそのままWebサーバーにアップロードしてアクセス。

---

## Step 7: テスト送信

1. 公開したフォームを開く
2. ダミーデータで申込を送信
3. スプレッドシートの「申込データ」シートを開く
4. データが追加されていることを確認

成功すれば、運用開始です 🎉

---

## 🛠️ 運営者の使い方（運用開始後）

スプレッドシート上部に「🏓 大会運営」メニューが追加されます。

| メニュー | 機能 |
|----------|------|
| 📋 初期設定 | 既に実施済み（やり直す時のみ） |
| 📊 集計表・名簿を再生成 | 全シートを最新状態に再構築 |
| 🔍 重複チェック | 同一選手の複数登録を検出 |
| 🧾 領収書を一括作成（チーム別） | 申込データから自動で全団体の領収書を発行 |
| 📝 任意領収書を発行 | 「領収書（任意発行）」シートで自由入力 |

### 任意領収書の使い方

1. 「📝 領収書（任意発行）」シートを開く
2. 上部の入力欄に：
   - 団体名・宛名
   - 金額
   - 但し書き（デフォルトで大会名が入っています）
   - 発行日
   - No.
3. メニュー → **🏓 大会運営** → **📝 任意領収書を発行**
4. 13行目以降に領収書プレビューが表示されます
5. 印刷範囲を選択して印刷

## ❓ よくあるトラブル

### Safariで日本語が文字化けする
→ HTMLファイルが UTF-8 で保存されているか確認してください。生成されたファイルは UTF-8 です。

### フォームから送信してもデータが反映されない
→ 以下を確認：
1. `GAS_URL` が正しく書き換えられているか
2. デプロイ時に「アクセスできるユーザー：全員」になっているか
3. ブラウザの開発者ツール（F12）の Network タブでエラーがないか

### 確認メールが届かない
→ Gmail の送信制限（無料アカウントは1日100通）に達している可能性があります。データはスプレッドシートに正常保存されているので問題ありません。

### スプレッドシートに「🏓 大会運営」メニューが出ない
→ スプレッドシートを再読み込み（F5）してください。

### GASコードを更新したい
→ 既存のデプロイを編集する場合：
1. デプロイ → デプロイを管理
2. 鉛筆アイコン → バージョン → **新バージョン**
3. **デプロイ**
（URLは同じまま、コードのみ更新されます）
